About This Project

Heart Attack Risk Predictor is a machine learning-based application designed to assess the risk of heart attacks using logistic regression.

This system analyzes various health parameters including:

  • Personal information (age, gender)
  • Vital signs (heart rate, blood pressure, blood sugar)
  • Cardiac biomarkers (CK-MB and Troponin levels)

The model has been trained on medical data to identify patterns associated with heart attack risk. By inputting your health metrics, the system provides a prediction to help raise awareness about cardiovascular health.
